*RESTful Web Service for Vehicle Entity*

-> I have implemented a restful web Service that performs CRUD operations for a vehicle entity using ASP DOT NET web application - Web API

Technologies Used:

1. Visual Studio 2015
2. C#
3. HTML
4. JQuery
5. JavaScript
6. LINQ
7. IIS (Internet Information Services Web Server)

Ways of Compiling and Runing the code:
-> I have used visual studio to run the code using Chrome Browser.
-> I have deployed the application in IIS server and then tested using Chrome Rest Client
